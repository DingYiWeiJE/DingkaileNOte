本篇题目：路灯照明

题目

在一条笔直的公路上安装了N个路灯，
从位置0开始安装，路灯之间间距固定为100米
每个路灯都有自己的照明半径
请计算第一个路灯和最后一个路灯之间，
无法照明的区间的长度和。

输入

第一行为一个数N，表示路灯个数，1 <= N <= 100000
第二行为N个空格分割的数，表示路灯的照明半径，1 <= 照明半径

输出描述

无法照明的区间的长度和。

示例一

输入

2
50 50

输出

0

示例二

输入

4
50 70 20 70

输出

20

说明

[170,180],[220,230]，两个未覆盖的区间，总里程为 20

代码编写逻辑

读取输入的整数 n，表示数组的长度
读取 n 个整数，存储在数组 arr 中
创建一个长度为(n-1)100 的布尔数组 bools，用于记录每个位置是否被覆盖
遍历数组 arr，对于每个元素 arr[i]，将以 i100 为中心，左右延伸 arr[i]个位置的区间内的所有位置都标记为已覆盖
统计 bools 数组中未被覆盖的位置个数，输出结果
具体来说，这段代码使用了 Java 8 中的 Stream API，通过 IntStream.range()方法生成一个整数范围，然后对于每个元素执行一个 Lambda 表达式。在这个 Lambda 表达式中，首先计算出当前元素对应的区间左右端点 left 和 right，然后将 bools 数组中 left 到 right-1 的所有元素都标记为已覆盖。使用 Stream API 中的 filter()方法和 count()方法统计 bools 数组中未被覆盖的位置个数，并输出结果。

Code

import java.util.Scanner;
import java.util.stream.IntStream;

public class Main {
  public static void main(String[] args) {
    try (Scanner s = new Scanner(System.in)) {
      int n = s.nextInt();
      int[] arr = new int[n];
      for (int i = 0; i < n; ++i) {
        arr[i] = s.nextInt();
      }
      boolean[] bools = new boolean[(arr.length - 1) * 100];

      IntStream.range(0, arr.length).forEach(i -> {
        int pos = i * 100;
        int left = Math.max(pos - arr[i], 0);
        int right = Math.min(pos + arr[i], bools.length);

        IntStream.range(left, right).forEach(k -> bools[k] = true);
      });

      long count = IntStream.range(0, bools.length).filter(i -> !bools[i]).count();
      System.out.println(count);
    }
  }
}
