流水线

题目

一个工厂有 m 条流水线，来并行完成 n 个独立的作业，该工厂设置了一个调度系统，在安排作业时，总是优先执行处理时间最短的作业。

现给定流水线个数 m，需要完成的作业数 n, 每个作业的处理时间分别为 t1,t2…tn。请你编程计算处理完所有作业的耗时为多少？

当 n>m 时，首先处理时间短的 m 个作业进入流水线，其他的等待，当某个作业完成时，依次从剩余作业中取处理时间最短的进入处理。

输入描述:

第一行为 2 个整数（采用空格分隔），分别表示流水线个数 m 和作业数 n；

第二行输入 n 个整数（采用空格分隔），表示每个作业的处理时长 t1,t2…tn。

0< m,n<100，0<t1,t2…tn<100。

注：保证输入都是合法的。

输出描述:

输出处理完所有作业的总时长

示例 1

输入

3 5
8 4 3 2 10

输出

13

说明

1、先安排时间为 2、3、4 的 3 个作业。

2、第一条流水线先完成作业，然后调度剩余时间最短的作业 8。

3、第二条流水线完成作业，然后调度剩余时间最短的作业 10。

4、总工耗时就是第二条流水线完成作业的时间 13（3+10）。

解题思路

第一步，使用 Scanner 类从控制台获取两行字符串，并使用 split()方法将字符串拆分成字符串数组，存储在变量 strs 和 jobs 中；

第二步，定义一个长度为 jobs.length 的整型数组 array，将 jobs 中的字符串转换成整数，存储在 array 中；

第三步，对 array 进行排序；

第四步，定义一个 List，将 array 中的元素按照 medValue 的值添加到 List 中；

第五步，对 List 进行排序；

第六步，输出 List 中最后一个元素的 sum 值。

主要使用了 ArrayList、Arrays 和 Collections 数据结构，考察了字符串处理、字符串拆分、字符串解析、排序、自定义比较器和 Lambda 表达式等知识点。

代码实现


import java.util.ArrayList;
import java.util.Arrays;
import java.util.Collections;
import java.util.List;
import java.util.Scanner;


public class Main {
	public static void sovle(String[] strs, String[] jobs) {
		int medValue = Integer.parseInt(strs[0]);
		int[] array = new int[jobs.length];
		for (int j = 0; j < jobs.length; j++) {
			array[j] = Integer.parseInt(jobs[j]);
		}
		// 排序
		Arrays.sort(array);
		List<MedicBean> myList = new ArrayList<>();
		for (int j = 0; j < array.length; ++j) {
			if (myList.size() < medValue) {
				myList.add(new MedicBean(array[j], array[j]));
			} else {
				// 找到数组中 total最小的medic 加进去
				Collections.sort(myList);
				MedicBean medic = myList.get(0);
				medic.sum += array[j];
			}
		}
		// 排序
		Collections.sort(myList);
		System.out.println(myList.get(myList.size() - 1).sum);
	}

	static class MedicBean implements Comparable {
		public int end;
		public int sum;

		public MedicBean(int end, int sum) {
			this.end = end;
			this.sum = sum;
		}

		@Override
		public int compareTo(Object obj) {
			MedicBean medic = (MedicBean) obj;
			return this.sum - medic.sum;
		}
	}

	public static void main(String[] args) {
		try {
			Scanner s = new Scanner(System.in);
			String[] strs = s.nextLine().split(" ");
			String[] jobs = s.nextLine().split(" ");
			sovle(strs, jobs);
		} catch (Exception e) {
			e.printStackTrace();
			System.out.println("input error");
		}
	}
}
