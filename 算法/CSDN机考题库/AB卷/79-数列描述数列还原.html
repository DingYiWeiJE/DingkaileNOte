本篇题目：数列还原

题目

有一个数列A[n]
从A[0]开始每一项都是一个数字
数列中A[n+1]都是A[n]的描述
其中A[0]=1
规则如下
A[0]:1
A[1]:11 含义其中A[0]=1是1个1 即11
表示A[0]从左到右连续出现了1次1
A[2]:21 含义其中A[1]=11是2个1 即21
表示A[1]从左到右连续出现了2次1
A[3]:1211 含义其中A[2]从左到右是由一个2和一个1组成 即1211
表示A[2]从左到右连续出现了一次2又连续出现了一次1
A[4]:111221 含义A[3]=1211 从左到右是由一个1和一个2两个1 即111221
表示A[3]从左到右连续出现了一次1又连续出现了一次2又连续出现了2次1
输出第n项的结果

输入

数列第n项
0 <= n <= 59

输出

数列内容

示例一

输入

4

输出

111221

代码编写逻辑

这个程序的实现逻辑是，先读入一个整数 n，然后从 1 开始，依次计算外观数列的前 n 项。在计算第 i 项时，先将第 i-1 项中连续的重复数字替换成数字和该数字的个数，得到第 i 项。重复这个过程 n 次，最后输出第 n 项。

具体实现时，使用 StringBuilder 来存储当前的外观数列，使用一个循环来计算外观数列的前 n 项。在循环中，先判断当前项是否为第一项，如果是，则直接输出 1；否则，对前一项中连续的重复数字进行替换，得到当前项，然后将当前项赋值给 StringBuilder，继续计算下一项。

Code

import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int n = Integer.parseInt(in.nextLine());
        in.close();

        StringBuilder content = new StringBuilder("1");

        if (n == 0) {
            System.out.println(content);
            return;
        }
        for (int i = 1; i <= n; i++) {
            StringBuilder next = new StringBuilder();

            char[] chars = content.toString().toCharArray();
            char last = chars[0];
            int count = 1;
            for (int j = 1; j < chars.length; j++) {
                if (chars[j] == last) count++;
                else {
                    next.append(count).append(last);
                    count = 1;
                    last = chars[j];
                }
            }
            next.append(count).append(last);
            content = next;
        }

        System.out.println(content);

    }
}
