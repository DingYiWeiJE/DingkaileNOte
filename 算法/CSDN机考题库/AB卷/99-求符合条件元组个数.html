OD 统一考试 B 卷：求符合条件元组个数

题目描述

给定一个整数数组 nums 、一个数字 k、一个整数目标值 target，请问 nums 中是否存在 k 个元素使得其相加结果为 target，请输出所有符合条件且不重复的 k 元组的个数

数据范围：

  • 2 < = n u m s . l e n g t h < = 200 2<= nums.length <= 200 2<=nums.length<=200
  • 1 0 9 < = n u m s [ i ] < = 1 0 9 10^9 <= nums[i] <= 10^9 109<=nums[i]<=109
  • 1 0 9 < = t a r g e t < = 1 0 9 10^9 <= target <= 10^9 109<=target<=109
  • 2 < = k < = 100 2 <= k <= 100 2<=k<=100

输入

第一行是 nums 取值：2 7 11 15

第二行是 k 的取值：2

第三行是 target 取值： 9

输出

输出第一行是符合条件的元祖个数： 1

说明

[2,7]满足，输出个数是 1

示例一

输入

-1 0 1 2 -1 -4
3
0

输出

2

说明

-1 0 1,-1 -1 2 满足条件

示例二

输入

2 7 11 15
2
9

输出

1

说明

2 7 符合条件

解题说明

通过回溯法找出数组 nums 中符合要求的 k 元组，使其相加结果为 target，并统计符合条件且不重复的 k 元组个数。

OD 统一考试 B 卷题解

import java.util.*;

public class Main {
    private static int res = 0;
    private static int target;
    private static Set<List<Integer>> numsSet = new HashSet<>();

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int[] nums = Arrays.stream(sc.nextLine().split(" ")).mapToInt(Integer::parseInt).toArray();
        int k = sc.nextInt();
        target = sc.nextInt();

        Arrays.sort(nums);
        combine(nums, k, new ArrayList<>(), 0, 0);

        System.out.println(res);
    }

    public static void combine(int[] nums, int n, List<Integer> list, int index, int sum) {
        if (n == 0) {
            if (sum == target) {
                numsSet.add(new ArrayList<>(list));
                res++;
            }
        } else {
            for (int i = index; i < nums.length; i++) {
                if (i > index && nums[i] == nums[i - 1]) {
                    continue;
                }
                if (sum + nums[i] > target) {
                    break;
                }
                list.add(nums[i]);
                combine(nums, n - 1, list, i + 1, sum + nums[i]);
                list.remove(list.size() - 1);
            }
        }
    }
}
