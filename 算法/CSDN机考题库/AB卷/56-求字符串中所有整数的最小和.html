求字符串中所有整数的最小和

题目

输入字符串 s，输出 s 中包含所有整数的最小和

说明

字符串 s，只包含 a-z A-Z ± ；

合法的整数包括

1） 正整数 一个或者多个 0-9 组成，如 0 2 3 002 102

2）负整数 负号 - 开头，数字部分由一个或者多个 0-9 组成，如 -0 -012 -23 -00023

输入描述:

包含数字的字符串

输出描述:

所有整数的最小和

示例 1

输入

bb1234aa

输出

10

示例 2

输入

bb12-34aa

输出

-31

解题思路

主要是使用了一个数据结构 ArrayList，用来存储输入字符串中的数字，然后求字符串中所有数字的和。该代码考察的集合和知识点主要包括：字符串的处理，ArrayList 的使用，算法的设计与实现。

代码实现


import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;


public class Main {
	public static void sovle(String values) {
        boolean fuNumber = false;
        String tempNumber = "-";
        List<Integer> myList = new ArrayList<>();
        for (int j = 0; j < values.length(); ++j) {
            char tem = values.charAt(j);
            if (tem == '-'){
                fuNumber = true;
                continue;
            }
            if (tem <= '9' && tem >= '0'){
                if (fuNumber){ //如果不是字符就一直往下记录
                    tempNumber += tem;
                }else {
                    myList.add(Integer.parseInt(String.valueOf(tem)));
                }
            }else {
                if (tempNumber != "-"){
                    myList.add(Integer.parseInt(tempNumber));
                    tempNumber = "-";
                }
                fuNumber = false;
            }
        }
        int sum = 0;
        for (int j = 0; j < myList.size(); ++j) {
            sum += myList.get(j);
        }
        System.out.println(sum);
	}

	public static void main(String[] args) {
		try {
			Scanner s = new Scanner(System.in);
	        String values = s.nextLine();
	        sovle(values);
		} catch (Exception e) {
			e.printStackTrace();
			System.out.println("input error");
		}
	}
}
