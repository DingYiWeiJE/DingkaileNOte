本篇题目：IPv4 地址转换成整数

题目

存在一种虚拟 IPv4 地址，由4小节组成，每节的范围为0~255，以#号间隔，
虚拟 IPv4 地址可以转换为一个32位的整数，例如：
128#0#255#255，转换为32位整数的结果为2147549183（0x8000FFFF）
1#0#0#0，转换为 32 位整数的结果为16777216（0x01000000）
现以字符串形式给出一个虚拟 IPv4 地址，限制第1小节的范围为1~128，
即每一节范围分别为(1~128)#(0~255)#(0~255)#(0~255)，
要求每个 IPv4 地址只能对应到唯一的整数上。
如果是非法 IPv4，返回invalid IP

  • 备注：
    输入不能确保是合法的 IPv4 地址，
    需要对非法 IPv4（空串，含有 IP 地址中不存在的字符，
    非合法的#分十进制，十进制整数不在合法区间内）进行识别，返回特定错误

输入

输入一行，虚拟 IPv4 地址格式字符串

输出

输出以上，按照要求输出整型或者特定字符

示例一

输入

100#101#1#5

输出

1684340997

说明

示例一

输入

1#2#3

输出

invalid IP

代码编写逻辑

下述代码的作用是将输入的 IP 地址转换为十进制表示。实现逻辑如下：

通过 Scanner 读取输入的 IP 地址，使用"#"分割成 4 个部分，存储在 ipParts 数组中。

判断 ipParts 数组的长度是否为 4，如果不是则输出"invalid IP"。

遍历 ipParts 数组，将每个部分转换为整数，如果转换失败或者整数不在 0~255 的范围内，则输出"invalid IP"。

将每个整数转换为 8 位二进制数，并将它们拼接成一个 32 位的二进制字符串。

将二进制字符串转换为十进制数，并输出。

需要注意的是，这段代码没有考虑 IP 地址的合法性，例如"256.0.0.0"这样的地址也会被转换为十进制数。如果需要判断 IP 地址的合法性，可以使用正则表达式或者其他方法进行判断。

Code

import java.util.Scanner;

public class Main {
    private static final String INVALID_IP = "invalid IP";

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        while (scanner.hasNextLine()) {
            String input = scanner.nextLine();
            String[] ipParts = input.split("#");

            if (ipParts.length != 4) {
                System.out.println(INVALID_IP);
                continue;
            }

            StringBuilder binaryString = new StringBuilder();
            for (String ipPart : ipParts) {
                try {
                    int num = Integer.parseInt(ipPart);
                    if (num < 0 || num > 255) {
                        System.out.println(INVALID_IP);
                        continue;
                    }
                    binaryString.append(String.format("%8s", Integer.toBinaryString(num)).replace(' ',
                                    '0'));
                } catch (NumberFormatException e) {
                    System.out.println(INVALID_IP);
                    continue;
                }
            }

            int decimalIp = Integer.parseInt(binaryString.toString(), 2);
            System.out.println(decimalIp);
        }
    }
}
