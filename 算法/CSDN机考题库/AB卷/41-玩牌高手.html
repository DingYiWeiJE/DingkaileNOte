本篇题解：玩牌高手

题目

给定一个长度为N的整数数组，表示一个选手在N轮内选择的牌面分数，
选手基于规则选牌，请计算所有轮结束后其可以获得的最高总分数。
选择规则如下：

  1. 在每轮里选手可以选择获取该轮牌面，则其总分数加上该轮牌面分数为其新的总分数
  2. 选手也可不选择本轮牌面，直接跳到下一轮，此时将当前总分数还原为 3 轮前的总分数，若当前轮次小于等于 3，则总分数置 0
  3. 选手初始总分数为 0，且必须依次参加每轮

输入

第一行为一个小写逗号分隔的字符串
表示N轮的牌面分数
1 <= N <= 20 分数值为整数 [-100,100]
不考虑格式问题

输出描述

所有轮结束后 选手能获得的最高总分数

示例一

输入

1,-5,-6,4,3,6,-2

输出

11

说明

总共7轮牌面，
第一轮选择该轮牌面总分数为1
第二轮不选择该轮牌面总分数还原为0
第三轮不选择该轮牌面总分数还原为0
第四轮选择该轮牌面总分数为4
第五轮选择该轮牌面总分数为7
第六轮选择该轮牌面总分数为13
第七轮如果不选择概论牌面 则总分数返回到三轮前的分数 即第四轮的总分数4
如果选择该轮牌面，总分数为11，所以选择该轮， 因此最高总分数为11

代码编写思路

本题是一个动态规划的例子，用于解决一个特定的问题。在这个问题中，我们需要找到一个数组中的一些数字，使得这些数字的和最大，但是不能选择相邻的数字。这个问题可以用动态规划来解决。

在下述代码中，我们定义了一个 solveMethod 方法，它接受两个参数：一个字符串数组和一个整数数组。字符串数组包含了输入的数字，整数数组是将字符串数组转换为整数后得到的结果。在方法中，我们定义了一个 dp 数组，用于存储到当前位置为止的最大和。我们首先将 dp[0]设置为 nums[0]，因为到第一个位置时，最大和就是 nums[0]本身。

然后我们遍历整个数组，对于每个位置，我们有三种选择：选择当前位置的数字，选择当前位置的数字和前面隔一个位置的数字，或者不选择当前位置的数字。我们需要选择这三种选择中的最大值，并将其存储在 dp 数组中。最后，我们输出 dp[nums.length-1]，即到最后一个位置时的最大和。

这个问题的解法比较经典，也比较重要，建议好好理解一下。

Code

import java.util.Scanner;

public class Main {
    public static void solveMethod(String[] values, int[] nums) {
        int[] dp = new int[nums.length];
        dp[0] = nums[0];
        for (int i = 1; i < nums.length; i++) {
            if (i == 1) {
                dp[i] = Math.max(dp[i - 1], nums[i]);
            } else if (i == 2) {
                dp[i] = Math.max(dp[i - 1], nums[i] + nums[i - 2]);
            } else {
                dp[i] = Math.max(dp[i - 1], dp[i - 3] + nums[i] + nums[i - 2]);
            }
        }
        System.out.println(dp[nums.length - 1]);
    }

    public static void main(String[] args) {
        Scanner s = new Scanner(System.in);
        String[] values = s.nextLine().split(",");
        int[] nums = new int[values.length];
        for (int i = 0; i < values.length; i++) {
            nums[i] = Integer.parseInt(values[i]);
        }
        solveMethod(values, nums);
    }
}

1,-5,-6,4,3,6,-2
11
