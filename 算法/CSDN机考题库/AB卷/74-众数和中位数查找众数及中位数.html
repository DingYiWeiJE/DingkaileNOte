本篇题目：众数和中位数

题目

  1. 众数是指一组数据中出现次数多的数
    众数可以是多个
  2. 中位数是指把一组数据从小到大排列，最中间的那个数，
    如果这组数据的个数是奇数，那最中间那个就是中位数
    如果这组数据的个数为偶数，那就把中间的两个数之和除以 2 就是中位数
  3. 查找整型数组中元素的众数并组成一个新的数组
    求新数组的中位数

输入

输入一个一维整型数组，数组大小取值范围 0 < n < 1000
数组中每个元素取值范围， 0 < e < 1000

输出描述

输出众数组成的新数组的中位数

示例一

输入

10 11 21 19 21 17 21 16 21 18 16

输出

21

示例二

输入

2 1 5 4 3 3 9 2 7 4 6 2 15 4 2 4

输出

3

示例三

输入

5 1 5 3 5 2 5 5 7 6 7 3 7 11 7 55 7 9 98 9 17 9 15 9 9 1 39

输出

7

解题思路

  1. 从控制台读入一个字符串 inputValues，其中包含多个整数，用空格隔开；
  2. 定义一个 Map 类型的 myMap，用于记录每个整数出现的次数；
  3. 将字符串 inputValues 按照空格分割成字符串数组 chars；
  4. 使用 Lambda 表达式遍历 chars 数组，将每个整数及其出现的次数存入 myMap 中；
  5. 找到 myMap 中出现次数最多的整数，记为 maxValue；
  6. 使用 Lambda 表达式对 myMap 的键值对进行排序，并将出现次数最多的整数存入 resultList 中；
  7. 根据 resultList 的大小判断中位数的位置，如果大小为偶数，则取中间两个数的平均值；如果为奇数，则取中间的数；
  8. 输出中位数。 该程序主要考察了 Map 和 List 的使用、Lambda 表达式和流式处理等知识点。

代码实现


import java.util.Arrays;
import java.util.HashMap;
import java.util.List;
import java.util.Map;
import java.util.Scanner;
import java.util.stream.Collectors;


public class Main {

	public static void main(String[] args) {
		try {
			Scanner s = new Scanner(System.in);
			String inputValues = s.nextLine();
			Map<Integer, Integer> myMap = new HashMap<>();
			String[] chars = inputValues.split(" ");
			Arrays.stream(chars).forEach(x -> {
				int i = Integer.parseInt(x);
				myMap.put(i, myMap.getOrDefault(i, 0) + 1);
			});
			Integer maxValue = myMap.values().stream().max(Integer::compareTo).get();
			List<Integer> resultList = myMap.keySet().stream().filter(k -> maxValue.equals(myMap.get(k)))
					.sorted(Integer::compareTo).collect(Collectors.toList());
			Integer result = 0;
			int size = resultList.size();
			if (resultList.size() % 2 == 0) {
				result = (resultList.get(size / 2) + resultList.get(size / 2 - 1)) / 2;
				System.out.print(result);
			} else {
				result = resultList.get(size / 2);
				System.out.print(result);
			}
		} catch (Exception e) {
			e.printStackTrace();
			System.out.println("input error");
		}
	}
}
