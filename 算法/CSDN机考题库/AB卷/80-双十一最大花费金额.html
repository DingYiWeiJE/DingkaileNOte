本篇题目：双十一

题目

双十一众多商品进行打折销售，小明想购买一些自己心仪的商品，
但由于受购买资金限制，所以他决定从众多心意商品中购买 3 件，
而且想尽可能的花完资金，
现在请你设计一个程序帮助小明计算尽可能花费的最大资金额。

输入

第一行为整型数组M，数组长度小于100，数组元素记录单个商品的价格；
单个商品价格小于1000；
第二行输入为购买资金的额度R；
R < 100000。

输出

输出为满足上述条件的最大花费额度
如果不存在满足上述条件的商品请返回-1

示例一

输入

23,26,36,27
78

输出

76

示例二

输入

23,30,40
26

输出

-1

代码编写逻辑

下述代码实现了一个求三个数之和的最大值的问题。具体实现逻辑如下：

读入一个字符串 m 和一个整数 r。
将字符串 m 按照逗号分隔符分割成一个字符串数组 goodsPricesString。
将 goodsPricesString 中的每个字符串转换成整数并存储在 goodsPrices 数组中。
对 goodsPrices 数组进行排序。
初始化一个变量 maxSum 为-1，用于存储三个数之和的最大值。
从头开始遍历 goodsPrices 数组，对于每个元素 goodsPrices[i]，在 i+1 到 goodsPrices.length-1 的范围内使用双指针法寻找另外两个数，使得三个数之和最大且不超过 r。
如果找到了一个三元组使得三个数之和不超过 r，就更新 maxSum 的值。

需要注意的是，下述代码中的双指针法是一个经典的算法，可以用于解决很多类似的问题。

Code

import java.util.*;

public class Main {
    public static void main(String[] args) {
        try (Scanner s = new Scanner(System.in)) {
            String m = s.nextLine();
            int r = s.nextInt();
            solveMethod(m, r);
        }
    }

    private static void solveMethod(String m, int r) {
        String[] goodsPricesString = m.split(",");
        int[] goodsPrices = new int[goodsPricesString.length];
        for (int i = 0; i < goodsPricesString.length; ++i) {
            goodsPrices[i] = Integer.parseInt(goodsPricesString[i]);
        }
        Arrays.sort(goodsPrices);
        int maxSum = -1;
        for (int i = 0; i < goodsPrices.length - 2; i++) {
            int j = i + 1;
            int k = goodsPrices.length - 1;
            while (j < k) {
                int sum = goodsPrices[i] + goodsPrices[j] + goodsPrices[k];
                if (sum <= r) {
                    maxSum = Math.max(maxSum, sum);
                    j++;
                } else {
                    k--;
                }
            }
        }
        System.out.println(maxSum);
    }
}
