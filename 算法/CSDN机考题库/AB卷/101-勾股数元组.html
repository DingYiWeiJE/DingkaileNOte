勾股数元组

如果 3 个正整数(a,b,c)满足 a2 + b2 = c2 的关系，则称(a,b,c)为勾股数（著名的勾三股四弦五），为了探索勾股数的规律，我们定义如果勾股数(a,b,c)之间两两互质（即 a 与 b，a 与 c，b 与 c 之间均互质，没有公约数），则其为勾股数元祖（例如(3,4,5)是勾股数元祖，(6,8,10)则不是勾股数元祖）。请求出给定范围[N,M]内，所有的勾股数元祖。

输入输出描述:

输入描述:

起始范围 N，1 <= N <= 10000

结束范围 M，N < M <= 10000

输出描述:

a,b,c 请保证 a < b < c,输出格式：a b c；

多组勾股数元祖请按照 a 升序，b 升序，最后 c 升序的方式排序输出；

给定范围中如果找不到勾股数元祖时，输出”NA”。

示例 1：

输入

1
20

输出

3 4 5
5 12 13
8 15 17

说明

[1, 20]范围内勾股数有：(3 4 5)，(5 12 13)，(6 8 10)，(8 15 17)，(9 12 15)，(12 16 20)；其中，满足(a,b,c)之间两两互质的勾股数元祖有：(3 4 5)，(5 12 13)，(8 15 17)；按输出描述中顺序要求输出结果。

示例 2：

输入

5
10

输出

NA

说明

[5, 10]范围内勾股数有：(6 8 10)；其中，没有满足(a,b,c)之间两两互质的勾股数元祖； 给定范围中找不到勾股数元祖，输出”NA”。

解题思路

我们输入数据，更具题目逻辑，找出所有勾股数，并且排除存在公约数的数组。它使用了布尔型数据结构来检查勾股数和公约数，使用了 ArrayList 和 Collections 数据结构来存储勾股数，使用了 while 循环来实现代码逻辑。

代码实现


import java.util.ArrayList;
import java.util.Collections;
import java.util.List;
import java.util.Scanner;



public class Main {
	private static boolean check(int i, int j, int max) {
		int mul = i * i + j * j;
		double res = Math.sqrt(mul);
		int sub = (int) res;
		return res * res == sub * sub && sub <= max && sub > j;
	}

	private static boolean checkGongYue(int i, int j, int k) {
		for (int i = 2; i < k; i++) {
			if (i % i == 0 && j % i == 0) {
				return true;
			} else if (i % i == 0 && k % i == 0) {
				return true;
			} else if (j % i == 0 && k % i == 0) {
				return true;
			}
		}
		return false;
	}

	public static void main(String[] args) {
		try {
			Scanner s = new Scanner(System.in);
			// [1,10000]
			int value1 = s.nextInt();
			// [n,10000]
			int value2 = s.nextInt();
			List<List<Integer>> myList = new ArrayList<>();
			for (int i = value1; i <= value2; ++i) {
				for (int j = i + 1; j <= value2; ++j) {
					if (check(i, j, value2)) {
						List<Integer> tempList = new ArrayList<>();
						tempList.add(i);
						tempList.add(j);
						tempList.add((int) Math.sqrt(i * i + j * j));
						Collections.sort(tempList);
						myList.add(tempList);
					}
				}
			}

			int index = 0;
			while (index < myList.size()) {
				List<Integer> list = myList.get(index);
				if (checkGongYue(list.get(0), list.get(1), list.get(2))) {
					myList.remove(index);
					index = 0;
				}
				index++;
			}
			for (int i = 0; i < myList.size(); ++i) {
				List<Integer> resultList = myList.get(i);
				System.out.println(resultList.get(0) + " " + resultList.get(1) + " " + resultList.get(2));
			}
		} catch (Exception e) {
			e.printStackTrace();
		}
	}
}
