本篇题目：运动会

题目

某学校举行运动会,学生们按编号(1、2、3.....n)进行标识,
现需要按照身高由低到高排列，
对身高相同的人，按体重由轻到重排列，
对于身高体重都相同的人，维持原有的编号顺序关系。
请输出排列后的学生编号

输入

两个序列，每个序列由 N 个正整数组成，(0 < n <= 100)。
第一个序列中的数值代表身高，第二个序列中的数值代表体重。

输出

排列结果，每个数据都是原始序列中的学生编号，编号从1开始。

示例一

输入

4
100 100 120 130
40 30 60 50

输出

2 1 3 4

示例二

输入

3
90 110 90
45 60 45

输出

1 3 2

代码编写逻辑

这段代码的主要逻辑是读取输入的学生数量，以及每个学生的身高和体重，然后将这些信息存储在一个二维数组中。接下来，使用 Java 内置的 Arrays.sort()方法对这个二维数组进行排序，排序的规则是先按照身高从小到大排序，如果身高相同则按照体重从小到大排序。最后，按照排序后的顺序输出每个学生的编号。

Code

import java.util.Arrays;
import java.util.Scanner;

public class Main {
  public static void main(String[] args) {
    try (Scanner s = new Scanner(System.in)) {
      int n = Integer.parseInt(s.nextLine());
      String heights = s.nextLine();
      String weights = s.nextLine();
      solveMethod(n, heights, weights);
    }
  }

  private static void solveMethod(int n, String heights, String weights) {
    String[] heightsArr = heights.split(" ");
    String[] weightsArr = weights.split(" ");
    int[][] ints = new int[n][3];
    for (int i = 0; i < n; ++i) {
      ints[i][0] = i + 1;
      ints[i][1] = Integer.parseInt(heightsArr[i]);
      ints[i][2] = Integer.parseInt(weightsArr[i]);
    }
    Arrays.sort(ints, (stu1, stu2) -> stu1[1] == stu2[1] ? stu1[2] - stu2[2] : stu1[1] - stu2[1]);

    for (int i = 0; i < n; ++i) {
      System.out.print(ints[i][0]);
      if (i != n - 1) {
        System.out.print(" ");
      }
    }
  }
}
