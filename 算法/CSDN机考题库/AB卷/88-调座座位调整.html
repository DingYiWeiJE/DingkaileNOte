华为 OD 统一考试（B 卷）题库：座位调整

题目描述

疫情期间课堂的座位进行了特殊的调整，不能出现两个同学紧挨着，必须隔至少一个空位。

给你一个整数数组 desk 表示当前座位的占座情况，由若干 0 和 1 组成，其中 0 表示没有占位，1 表示占位。在不改变原有座位秩序情况下，还能安排坐几个人？

输入

第一行是个子数组表示作为占座情况，由若干 0 和 1 组成，其中 0 表示没有占位，1 表示占位

输出

输出数值表示还能坐几个人

说明

1 < = d e s k . l e n g t h < = 2 ∗ 1 0 4 1 <= desk.length <= 2 * 10^4 1<=desk.length<=2∗104

示例一

输入

1,0,0,0,1

输出

1

示例二

输入

0,0,0,0,0

输出

3

说明

只有 desk[2] 的位置可以坐一个人

思路解析

该题的目的是计算连续出现的数字 “1” 的配对数，通过逐个遍历数组元素并进行条件判断，以确定连续序列的开始、结束和中间元素，并根据计数情况计算配对数。

华为 OD 统一考试（B 卷）题库题解

import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String[] desks = scanner.nextLine().split(",");
        int length = desks.length;
        double count  = 0;
        int pairsCount = 0;

        for (String num : desks) {
            if (num.equals("1")) {
                if (count  > 1) {
                    pairsCount += Math.ceil((count  - 2) / 2);
                }
                count  = 0;
            } else {
                if (count  == 0 || count  == length - 1) {
                    count ++;
                }
                count ++;
            }
        }

        if (count  > 1) {
            pairsCount += Math.ceil((count  - 2) / 2);
        }

        System.out.println(pairsCount);
    }
}
