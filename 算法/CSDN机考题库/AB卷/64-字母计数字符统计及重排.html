本篇题目：字母计数

题目

给出一个只包含字母的字符串,
不包含空格,统计字符串中各个子字母(区分大小写)出现的次数,
并按照字母出现次数从大到小的顺序输出各个字母及其出现次数
如果次数相同,按照自然顺序排序,且小写字母在大写字母之前

输入

输入一行仅包含字母的字符串

输出

按照字母出现次数从大到小的顺序输出各个字母和字母次数,
用英文分号分割,
注意末尾的分号
字母和次数中间用英文冒号分隔

示例一

输入

xyxyXX

输出

x:2;y:2;X:2;

说明

每个字符出现的次数为2 故x排在y之前
而小写字母x在大写X之前

示例二

输入

abababb

输出

b:4;a:3;

说明

b的出现个数比a多 故排在a前、

解题思路

  1. 从控制台读入一个字符串 inputStr；
  2. 定义一个 Map 类型的 charNumMap，用于记录每个字符出现的次数；
  3. 将字符串转换成字符数组 wordChars；
  4. 遍历字符数组，将每个字符及其出现的次数存入 charNumMap 中；
  5. 对 charNumMap 中的元素按照出现次数和字符的 ASCII 码进行排序；
  6. 输出排序后的结果。 该程序主要考察了字符串和字符数组的基本操作、Map 和 List 的使用、Lambda 表达式和流式处理等知识点。

代码实现


import java.util.HashMap;
import java.util.List;
import java.util.Map;
import java.util.Scanner;
import java.util.stream.Collectors;


public class Main {

	public static void main(String[] args) {
		try {
			Scanner s = new Scanner(System.in);
			String inputStr = s.nextLine();
			Map<Character, Integer> charNumMap = new HashMap<>();
			char[] wordChars = inputStr.toCharArray();
			for (char c : wordChars) {
				charNumMap.put(c, charNumMap.getOrDefault(c, 0) + 1);
			}
			List<Map.Entry<Character, Integer>> resultList = charNumMap.entrySet().stream().sorted((w1, w2) -> {
				char char1 = w1.getKey();
				char char2 = w2.getKey();
				int num1 = w1.getValue();
				int num2 = w2.getValue();
				if (num1 != num2) {
					return Integer.compare(num2, num1);
				}
				if ((char2 < 95 && char1 < 95) || (char2 > 95 && char1 > 95)) {
					return char1 - char2;
				} else {
					return char2 - char1;
				}
			}).collect(Collectors.toList());
			for (Map.Entry<Character, Integer> e : resultList) {
				System.out.print(e.getKey() + ":" + e.getValue() + ";");
			}
		} catch (Exception e) {
			e.printStackTrace();
			System.out.println("input error");
		}
	}
}
